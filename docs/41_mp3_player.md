## MP3 Player
<!-- Needs a screenshot of the MP3 player -->
The N64FlashcartMenu includes an MP3 Player that can read MP3 and FLAC files from the SD Card. MP3 sound files should have a sample rate of 44100 Hz or higher.

Whilst in the menu, select a music file in the File Browser to go to the MP3 Player screen, where the audio playback will begin immediately. Embedded cover art and metadata are displayed when available.

Controls:
- `A` Button: pause and resume playback
- `B` Button: stop playback and return to the File Browser
- `Left`/`Right` directional or Control Stick: seek within the current track
- `Up`/`Down` directional: skip to the previous or next track
- `R` Button: cycle playback mode (Normal, Loop, Repeat One, Shuffle, Party)
